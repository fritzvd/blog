Making sure GDAL works with ECW libecw under linux

This took me a while to unravel, so before I forget or before other people run into the same issues, let’s make a note of this.

Assuming you are using a debian-like linux distribution (ubuntu, debian, linux mint, etc. anything using: apt-get to install packages) there are a few things to note.

First of all make sure you have the latest packages for: mapserver, gdal, etc.

(ubuntu-gis repository has newer versions than the standard repositories).

The ecw library version 3.3 was free to use with a few exceptions. Read write access is ensured, write access up to files of 500MB. The first requirement is to build this library. Go to: this place to download libecwj2-3.3.20060906.zip. Untar and build the library.
