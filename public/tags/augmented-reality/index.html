Different Directions

October 2, 2010

It seems that my minor thesis will focus more on the user experience of augmented reality. The EIA’s have raster outputs that can be converted to 3D spaces. These spaces can then be given Locations. The Raster is already georeferenced which would make this easier (I think). The best method of viewing these things would seem Augmented Reality browsers such as Layar or Junaio. So I would have to brush up a bit on what is possible in which browser.
